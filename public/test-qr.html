Troubleshooting

If you're having trouble connecting from your phone:

  1. Make sure your phone and computer are on the same WiFi network
  2. Temporarily disable Windows Firewall or antivirus
  3. Try using a tunneling service like ngrok
  4. Check that your Vite server is running with host: '0.0.0.0'

Alternative: Use ngrok for Testing

If direct connection doesn't work, consider using ngrok:

  1. Install ngrok: npm install -g ngrok
  2. Start your Vite server
  3. In another terminal: ngrok http 5173
  4. Use the provided ngrok URL on your phone
